# Numerical Methods in MATLAB

This repository contains MATLAB scripts implementing various numerical methods for solving differential equations and linear algebra problems. The methods and examples included here are based on the content studied from the book included in this repository.

## Contents

- Finite difference methods for PDEs such as the heat equation and wave equation
- Iterative methods for solving linear systems (Gauss-Seidel, Jacobi)
- LU and Cholesky decompositions for matrix factorization and solving linear equations
- Runge-Kutta methods for ODEs
- Comparison between numerical and exact solutions where applicable

## Usage

Each MATLAB script (.m file) contains detailed comments and explanations of the implemented method. You can run them individually in MATLAB to explore the behavior of the numerical methods.

## About

These codes were developed as part of coursework to deepen understanding of numerical methods for solving mathematical physics problems.
